# Kittygram Frontend

Frontend приложение для Kittygram.

## Установка зависимостей

```bash
npm install
```

## Запуск в режиме разработки

```bash
npm start
```

Приложение будет доступно по адресу [http://localhost:3000](http://localhost:3000)

## Сборка для production

```bash
npm run build
```

## Запуск тестов

```bash
npm test
```
